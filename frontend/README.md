# SmartLogix - Frontend

Esta es la aplicación web desarrollada en React, estructurada como un componente NPM. Consumirá datos exclusivamente del BFF (Backend For Frontend).

## Requisitos
- Node.js 18+
- npm o yarn

## Instalación

```bash
npm install
```

## Ejecución

Para iniciar el servidor de desarrollo localmente:
```bash
npm start
```
La aplicación correrá en `http://localhost:3000`.

## Pruebas
Para ejecutar las pruebas del componente React:
```bash
npm test
```
